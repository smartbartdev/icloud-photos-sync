# Roadmap

## v0.2 candidates

- Improve scan visibility for very large libraries (explicit scan progress counters)
- Add optional safeguard for assets with missing/invalid `created_at`
- Improve Homebrew formula portability (cross-arch dependency handling)

## v0.3 candidates

- Add optional selective sync by album and/or year range
- Add import/export support for sync metadata diagnostics
- Add richer doctor checks (network/auth diagnostics mode)

## Backlog ideas

- Release automation for tap formula dependency refresh
- Optional telemetry-free debug bundle export command
- Documentation website and FAQ
